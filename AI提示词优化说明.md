# AI 提示词优化说明

## 📋 优化概览

已成功优化 `src/utils/aiService.js` 中的 AI 提示词，将生成的心理测评报告从**专业化、学术化**的风格转变为**温暖、通俗、有共鸣感**的朋友式交流风格。

---

## 🎯 核心改进

### 1. **角色定位转变**

**优化前：**
```
你是一位专业的心理咨询师，擅长社交焦虑障碍的评估。
```

**优化后：**
```
你是一位温暖、善解人意的心理陪伴者，就像用户最信任的朋友或贴心的心理咨询师。
你的任务是用最温暖、最通俗的语言，帮助用户理解自己的社交焦虑模式。
```

**改进点：**
- 从"专业评估者"转变为"温暖陪伴者"
- 强调"朋友"角色，而非"医生"角色
- 明确要求使用通俗语言

---

### 2. **新增核心语言风格指导**

**新增内容：**
```
【重要：语言风格要求】
🎯 核心原则：像朋友聊天，不是写专业报告
- 想象你正在和一位好朋友面对面聊天，用日常对话的语气
- 完全避免心理学术语（如"障碍"、"症状"、"病理"、"认知偏差"等）
- 用生活化的比喻和场景来解释，而不是理论概念
- 让用户感到"被看见"、"被理解"，而不是"被诊断"
- 语气要温柔、接纳、充满希望，像在说"我懂你的感受"
```

**改进点：**
- 明确禁止使用专业术语
- 要求使用生活化比喻
- 强调情感共鸣而非客观分析

---

### 3. **类型名称优化**

**优化前：**
```
示例："脑内彩排型社恐"、"完美主义表演者"、"情绪雷达过载型"
```

**优化后：**
```
示例："脑内彩排一百遍星人"、"社交电池秒没电型"、"人群中的隐形人"
要让用户看到后会心一笑，觉得"这就是我啊！"
```

**改进点：**
- 更加生活化、网络化的表达
- 增加趣味性和亲切感
- 强调用户的情感共鸣

---

### 4. **核心特征描述优化**

**新增对比示例：**

❌ **不要写：**
```
"在社交场景中表现出显著的预期焦虑"
"对负面评价具有高度敏感性"
```

✅ **要写：**
```
"还没到聚会现场，你就已经在脑子里把可能发生的尴尬场景演了一遍又一遍"
"别人一个不经意的眼神，你都能脑补出一部'他是不是觉得我很奇怪'的连续剧"
```

**改进点：**
- 用具体场景替代抽象概念
- 用第二人称"你"增加亲近感
- 用生动的比喻（"脑补连续剧"）
- 让用户产生"这说的就是我"的感觉

---

### 5. **"心理根源分析"改名为"为什么会这样"**

**优化前：**
```
4. 心理根源分析（2-3个维度，每个包含标题和详细说明）
   - 标题：8-12字，点出根本原因
   - 说明：40-60字，结合心理学理论深入分析
   - 要有深度，但不要过于学术化
```

**优化后：**
```
4. 为什么会这样（2-3个原因，每个包含标题和说明）
   ⚠️ 注意：不要叫"心理根源"，这太专业了！
   
   ✨ 标题要求（8-12字）：
   - 用温柔、理解的语气，像在说"我理解你为什么会这样"
   - 示例："你只是太想做到完美"、"小时候的经历在影响你"、"你的大脑太善于保护你了"
   
   ✨ 说明要求（50-80字）：
   - 完全不要用心理学术语！
   - 用生活化的比喻来解释
```

**新增对比示例：**

❌ **不要写：**
```
"这源于负强化机制导致的回避行为固化"
```

✅ **要写：**
```
"就像你小时候被狗吓过一次，之后见到狗都会绕道走。每次你逃避社交，短期内确实轻松了，
但大脑就记住了'逃避=安全'这个公式，下次就更想逃了。"
```

**改进点：**
- 标题从专业术语改为温柔理解的语气
- 完全禁止心理学术语
- 用日常生活比喻（被狗吓过）解释心理机制
- 语气像朋友在解释，而非专家在分析

---

### 6. **"正向重构"改名为"换个角度看自己"**

**优化前：**
```
5. 正向重构（60-80字）
   - 将社恐特质转化为潜在优势
   - 给予希望和力量
   - 温暖而不敷衍
```

**优化后：**
```
5. 换个角度看自己（60-80字）
   ⚠️ 不要叫"正向重构"，太专业！
   
   ✨ 语言风格要求：
   - 像朋友在鼓励你："你知道吗，你这个特点其实也有好的一面"
   - 真诚地指出这些特质的积极面，不要空洞的鸡汤
   - 给予具体的希望，而不是泛泛的安慰
```

**新增对比示例：**

❌ **不要写：**
```
"你的高敏感性可以转化为共情能力优势"
```

✅ **要写：**
```
"你对别人情绪的敏感，其实说明你是个很细腻、很会照顾别人感受的人。
这种特质用对地方，会让你成为朋友圈里最温暖、最懂人心的那个人。"
```

**改进点：**
- 标题更加口语化
- 避免"优势转化"等专业表达
- 用具体场景（"朋友圈里最温暖的人"）而非抽象概念
- 语气真诚，避免空洞鸡汤

---

### 7. **新增语言风格总结清单**

**新增内容：**
```
【语言风格总结 - 请务必遵守】
✅ 要用的表达：
- "你是不是经常..."、"每次...的时候"
- "就像..."（用比喻）
- "其实啊..."、"你知道吗..."
- "我懂那种感觉..."
- 具体的场景描述

❌ 绝对不要用的词汇：
- 障碍、症状、病理、疾病
- 认知偏差、负强化、泛化
- 功能损害、临床表现
- 任何教科书式的专业术语
```

**改进点：**
- 提供具体的语言模板
- 明确列出禁用词汇清单
- 便于 AI 快速理解和遵守

---

### 8. **特殊情况处理优化**

**优化前：**
```
- 如果总分较高（>80），要更关注功能损害和专业建议
```

**优化后：**
```
【特殊情况处理】
- 如果总分很高（>80），要温柔地建议寻求专业帮助，但不要吓唬用户
- 用"找个专业的心理咨询师聊聊，可能会帮到你"，而不是"需要临床干预"
```

**改进点：**
- 避免"功能损害"等专业术语
- 提供具体的温柔表达方式
- 强调不要吓唬用户

---


**改进点：**
- 强化"朋友"而非"专家"的角色
- 明确"通俗易懂"、"有温度"的要求
- 强调"被理解、被接纳"的情感目标

## 📊 优化效果对比

### 优化前的报告风格：
```
【类型名称】评价敏感型社恐
【核心特征】
- 在社交场景中表现出显著的预期焦虑
- 对负面评价具有高度敏感性
- 社交后反刍思维明显

【心理根源】
- 标题：自我价值外化
- 说明：将自我价值完全建立在他人的评价之上，缺乏稳定的内在价值感，导致对外部评价极度敏感。
```

**问题：**
- ❌ 专业术语过多（"预期焦虑"、"反刍思维"、"自我价值外化"）
- ❌ 语气客观冷静，缺乏温度
- ❌ 像在读诊断报告，而非朋友交流

---

### 优化后的报告风格：
```
【类型名称】别人眼光放大镜星人
【核心特征】
- 还没到聚会现场，你就已经在脑子里把可能发生的尴尬场景演了一遍又一遍
- 别人一个不经意的眼神，你都能脑补出一部"他是不是觉得我很奇怪"的连续剧
- 社交结束后，你会反复回想自己说的每句话，越想越觉得"我当时怎么那么傻"

【为什么会这样】
- 标题：你只是太在意别人的看法了
- 说明：就像考试前总担心考不好一样，你特别害怕在别人眼里"表现不好"。可能小时候，你的表现总是被拿来和别人比较，或者被批评过，所以现在你特别敏感别人怎么看你。
```

**优化效果：**
- ✅ 完全避免专业术语
- ✅ 用生活化场景和比喻
- ✅ 语气温暖、理解、像朋友在说话
- ✅ 用户容易产生"这说的就是我"的共鸣

---

## 🎉 总结

通过这次优化，AI 生成的心理测评报告将：

1. **更有温度** - 像朋友聊天，而非专业诊断
2. **更易理解** - 完全避免术语，用日常语言和比喻
3. **更有共鸣** - 用具体场景让用户感到"被看见"
4. **更有希望** - 温柔鼓励，而非冰冷分析

这将大大提升用户体验，让测评报告真正成为一份温暖的心理陪伴，而不是冷冰冰的专业文档。

## 🎯 整体优化效果

通过这两轮优化，整个报告从头到尾都保持了一致的温暖、通俗、有共鸣感的风格：

### ✅ 统一的语言风格
- **AI 生成部分**：像朋友在聊天
- **固定文案部分**：同样温暖亲切
