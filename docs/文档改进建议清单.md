```
- 使用连字符分隔：API-Reference.md
- 英文命名，便于跨平台
- 见名知义，避免缩写
```

### 文档结构模板
```markdown
# 标题

> **更新日期**: YYYY-MM-DD
> **适用版本**: v1.2.0+
> **难度级别**: 初级/中级/高级

## 目录
[自动生成或手动维护]

## 概述
[一句话说明文档用途]

## 主要内容
[详细内容]

## 相关文档
- [链接1](path)
- [链接2](path)

## 常见问题
Q: ...
A: ...
```
