## 🚢 生产部署

### 构建命令
```bash
npm run build
```

### 部署文件
构建后会生成 `dist/` 目录，包含所有静态文件。

### 部署方式
- **静态托管**: Vercel、Netlify、GitHub Pages
- **云服务器**: Nginx、Apache
- **CDN**: 阿里云OSS、腾讯云COS

### 配置要点
- 需要配置HTML5 History模式路由
- 推荐使用HTTPS
- 建议配置gzip压缩

当前为开发阶段，**任何格式正确的激活码都可通过验证**。
