### 2️⃣ **答题选项分数顺序完全打乱**

**原问题：** 所有题目都是按照1-2-3-4-5的固定顺序，容易产生：
- 应答定势（形成思维惯性）
- 作答疲劳
- 降低测评有效性

**优化方案：** 每道题的选项分数都进行了随机打乱

**示例对比：**

```javascript
// ❌ 优化前：固定顺序
{
  question: '需要参加多人聚会时，你的第一反应是？',
  options: [
    { id: 1, text: '期待见到朋友，感到开心', score: 1 },
    { id: 2, text: '有些紧张但愿意参加', score: 2 },
    { id: 3, text: '明显焦虑，需要心理准备', score: 3 },
    { id: 4, text: '非常抗拒，想找理由推脱', score: 4 },
    { id: 5, text: '绝对不会参加', score: 5 }
  ]
}

// ✅ 优化后：打乱顺序
{
  question: '需要参加多人聚会时，你的第一反应是？',
  options: [
    { id: 1, text: '非常抗拒，想找理由推脱', score: 4 },
    { id: 2, text: '期待见到朋友，感到开心', score: 1 },
    { id: 3, text: '明显焦虑，需要心理准备', score: 3 },
    { id: 4, text: '绝对不会参加', score: 5 },
    { id: 5, text: '有些紧张但愿意参加', score: 2 }
  ]
}
```

**效果：**
- ✅ 用户需要认真阅读每个选项
- ✅ 避免机械式作答
- ✅ 提高测评质量

### 4️⃣ **新增核心维度详解**

#### 🆕 负面评价恐惧（Fear of Negative Evaluation）

**为什么重要：**
- 这是社交焦虑的**核心病理特征**
- 国际主流量表（LSAS、SPIN）都包含此维度
- 比单纯的"场景恐惧"更能揭示深层心理

**题目示例：**
- Q14: 在社交场合中，你多担心"别人觉得我很奇怪"？
- Q15: 当你意识到别人在看你时，你会？
- Q16: 你害怕在公共场合出丑或尴尬吗？
- Q17: 如果需要在众人面前吃东西或喝水，你会？
- Q18: 你担心自己的穿着、外貌被别人评价吗？

#### 🆕 功能损害程度（Functional Impairment）

**为什么重要：**
- 评估社交焦虑对**实际生活的影响**
- 区分"正常紧张"和"需要干预"的重要指标
- 临床诊断的必要条件

**题目示例：**
- Q26: 社交焦虑对你的工作/学习造成了多大影响？
- Q27: 因为社交焦虑，你是否错过了重要的机会？
- Q28: 社交焦虑对你的人际关系影响有多大？
- Q29: 你是否因为社交焦虑而感到生活质量明显下降？

#### 🆕 效度检验题（Validity Check）

**为什么重要：**
- 识别**无效作答**（随便选的）
- 识别**社会称许性偏差**（故意选"正常"答案）
- 提高测评结果的可靠性

**题目设计：**
- Q34: 过去一年里，你在社交场合中是否从未感到过一丝紧张或不适？
  - 如果选"是的，我从未紧张过" → 标记为疑似无效（score: -999）
  
- Q35: 你是否认真阅读了每道题目并根据真实情况作答？
  - 如果选"随便选的" → 标记为低质量作答（score: -888）

### 2. scoring.js（计分算法）

**需要修改的地方：**

```javascript
// 1. 处理效度检验
function checkValidity(answers) {
  const q34 = answers[34]
  const q35 = answers[35]
  
  let isValid = true
  let warnings = []
  
  // 检查Q34：从未紧张过（不符合实际）
  if (q34 === -999) {
    warnings.push('检测到疑似无效作答（Q34）')
    isValid = false
  }
  
  // 检查Q35：随便选的
  if (q35 === -888) {
    warnings.push('检测到低质量作答（Q35）')
    isValid = false
  }
  
  return { isValid, warnings }
}

// 2. 按维度计算分数
function calculateDimensionScores(answers) {
  const dimensions = {
    scene_fear: { questions: [1,2,3,4,5], maxScore: 25 },
    avoidance: { questions: [6,7,8,9], maxScore: 20 },
    anticipation: { questions: [10,11,12,13], maxScore: 20 },
    fear_of_negative_evaluation: { questions: [14,15,16,17,18], maxScore: 25 },
    rumination: { questions: [19,20,21,22], maxScore: 20 },
    physical: { questions: [23,24,25], maxScore: 15 },
    functional_impairment: { questions: [26,27,28,29], maxScore: 20 },
    self_efficacy: { questions: [30,31,32,33], maxScore: 20 }
  }
  
  const scores = {}
  
  for (const [dim, config] of Object.entries(dimensions)) {
    let totalScore = 0
    for (const qId of config.questions) {
      totalScore += answers[qId] || 0
    }
    
    scores[dim] = {
      raw: totalScore,
      percentage: (totalScore / config.maxScore * 100).toFixed(1)
    }
  }
  
  return scores
}

// 3. 计算总分（排除效度题）
function calculateTotalScore(answers) {
  let total = 0
  
  // Q1-Q33计分，Q34-Q35不计分
  for (let i = 1; i <= 33; i++) {
    const score = answers[i]
    if (score > 0) { // 排除效度题的负分
      total += score
    }
  }
  
  return total
}

// 4. 生成报告
export function generateReport(answers, basicInfo) {
  // 先检查效度
  const validity = checkValidity(answers)
  
  if (!validity.isValid) {
    return {
      isValid: false,
      warnings: validity.warnings,
      message: '检测到作答质量问题，建议重新认真作答'
    }
  }
  
  // 计算分数
  const totalScore = calculateTotalScore(answers)
  const dimensionScores = calculateDimensionScores(answers)
  const level = determineLevel(totalScore)
  
  return {
    isValid: true,
    basicInfo, // 包含用户的基础信息
    totalScore,
    dimensionScores,
    level,
    recommendations: generateRecommendations(level, dimensionScores)
  }
}
```
