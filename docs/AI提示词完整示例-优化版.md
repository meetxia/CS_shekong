# AI提示词完整示例 - 优化版（包含题目和答案）

## 📊 优化说明

### 🎯 优化目标
让AI看到用户在**具体场景**下的反应，而不只是冰冷的分数，从而生成更精准的个性化分析。

### ✨ 主要改进
1. **发送题目内容**：AI能看到"需要参加多人聚会时"这样的具体场景
2. **发送用户选择**：AI能看到"非常抗拒，想找理由推脱"这样的具体反应
3. **智能筛选**：只发送焦虑程度较高的题目（分数≥3），最多12个
4. **隐私保护**：所有题目都是标准化量表，不涉及个人身份信息

---

## 📝 完整的AI提示词示例

```
你是一位专业的心理咨询师，擅长社交焦虑障碍的评估。请根据以下测评数据，为用户生成一份深度个性化的社恐类型分析报告。

【用户基本信息】
年龄段: 18-22岁（大学生）
性别: 女
职业: 学生
社交频率: 1-2次/周

【测评结果】
总分: 65/100
等级: 中度社交焦虑
初步类型: 预演型社恐

【维度得分详情】
社交场景恐惧: 18/25 (72%) - 偏高
回避行为程度: 15/20 (75%) - 偏高
预期焦虑强度: 16/20 (80%) - 偏高
负面评价恐惧: 17/25 (68%) - 中高
社交后反刍: 19/20 (95%) - 偏高
功能损害程度: 14/20 (70%) - 中高

【高焦虑题目】
Q1, Q3, Q6, Q10, Q14, Q19, Q24, Q26

【用户答题详情】
以下是用户在关键题目上的具体选择，请结合这些具体场景分析：

Q1. 需要参加多人聚会时，你的第一反应是？
   用户选择：非常抗拒，想找理由推脱

Q3. 在电梯或狭小空间里遇到陌生人时，你的感受是？
   用户选择：极度焦虑，想立刻离开

Q6. 收到社交邀请（聚会、活动等）时，你通常会？
   用户选择：几乎总是拒绝

Q10. 知道明天要参加社交活动，今晚你会？
   用户选择：焦虑到难以入睡

Q14. 在社交场合中，你多担心"别人觉得我很奇怪"？
   用户选择：极度担心，这是我最大的恐惧

Q19. 社交活动结束后，你会？
   用户选择：不断懊悔说错的话

Q24. 社交时，你的呼吸和肌肉状态是？
   用户选择：明显呼吸不顺，肌肉紧张

Q26. 社交焦虑对你的工作/学习造成了多大影响？
   用户选择：中度影响，有些任务难以完成

【任务要求】
请生成以下内容（JSON格式）：

1. **个性化社恐类型名称**（15字以内，要新颖、精准、有共鸣感）
   - 不要用常规词汇，要根据用户的具体表现创造一个独特的类型名
   - 示例："脑内彩排型社恐"、"完美主义表演者"、"情绪雷达过载型"

2. **英文名称**（体现专业性）

3. **核心特征**（3-5条，每条20-30字）
   - 要具体、形象，让用户有"这说的就是我"的感觉
   - 基于维度得分的具体表现

4. **心理根源分析**（2-3个维度，每个包含标题和详细说明）
   - 标题：8-12字，点出根本原因
   - 说明：40-60字，结合心理学理论深入分析
   - 要有深度，但不要过于学术化

5. **正向重构**（60-80字）
   - 将社恐特质转化为潜在优势
   - 给予希望和力量
   - 温暖而不敷衍

【注意事项】
- 语言要温暖、共情、专业，但不要说教
- 避免模板化表达，要针对这个用户的独特模式
- 不要过度强调"病态"，而是理解和接纳
- 如果总分较高（>70），要更关注功能损害和专业建议

【输出格式】
请严格按照以下JSON格式输出，不要有任何其他文字：

{
  "typeName": "你的个性化类型名称",
  "englishName": "Personalized Type Name",
  "features": [
    "核心特征1",
    "核心特征2",
    "核心特征3"
  ],
  "rootCauses": [
    {
      "title": "根源1标题",
      "desc": "根源1详细说明"
    },
    {
      "title": "根源2标题",
      "desc": "根源2详细说明"
    }
  ],
  "positiveReframe": "正向重构内容"
}
```

## 💡 优化效果对比

### ❌ 之前（只有分数）
```
高焦虑题目: Q1, Q3, Q6, Q10, Q14, Q19, Q24, Q26
```
**AI的理解**：用户在这些题目上分数很高，但不知道具体是什么场景

### ✅ 现在（包含题目和答案）
```
Q1. 需要参加多人聚会时，你的第一反应是？
   用户选择：非常抗拒，想找理由推脱

Q3. 在电梯或狭小空间里遇到陌生人时，你的感受是？
   用户选择：极度焦虑，想立刻离开
```
**AI的理解**：用户对多人聚会和封闭空间特别恐惧，可以生成"聚会回避+空间焦虑型社恐"这样精准的类型

## 📊 预期收益

### 1. 更精准的类型命名
- **之前**：预演型社恐（通用）
- **现在**：聚会恐惧+预演焦虑型社恐（具体）

### 2. 更贴合的特征描述
- **之前**：你在社交前会感到焦虑（泛泛而谈）
- **现在**：你对多人聚会特别抗拒，在封闭空间遇陌生人会极度焦虑（具体场景）

### 3. 更针对性的建议
- **之前**：建议从小型社交开始练习（通用建议）
- **现在**：建议先练习在开放空间与1-2人交流，逐步适应多人环境（针对性建议）

## 🔐 隐私保护

### ✅ 会发送的信息
1. 匿名化的基本信息（年龄段、性别、职业类别）
2. 测评分数和维度得分
3. 题目内容和用户选择（标准化量表）
4. 最多12个高焦虑题目

### ❌ 不会发送的信息
1. 用户的真实姓名
2. 联系方式（手机号、邮箱）
3. IP地址、设备信息
4. 用户自己填写的文字内容
5. 任何可识别个人身份的信息

### 🛡️ 安全性说明
- 所有题目都是标准化心理测评题，不涉及个人隐私
- 即使AI API被攻击，也无法追溯到具体个人
- 用户数据存储在本地浏览器，不上传服务器

## ✅ 总结

这次优化让AI能够：
1. 看到用户在**具体场景**下的反应（如"电梯遇陌生人"）
2. 理解用户的**真实感受**（如"极度焦虑，想立刻离开"）
3. 生成**更精准**的个性化分析
4. 给出**更有针对性**的改善建议

同时仍然保护用户隐私，不发送任何个人身份信息。

**这是一个在隐私保护和分析精准度之间的最佳平衡！** ✨
