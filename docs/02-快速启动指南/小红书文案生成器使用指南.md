# 小红书文案生成器使用指南

## 功能简介

在管理员后台新增了**小红书文案生成器**功能，可以通过AI快速生成社恐测评网站的推广文案，包括标题、正文、标签等，并支持一键复制到小红书发布。

## 快速开始

### 1. 访问入口

登录管理员后台后，点击左侧导航栏的 **📝 小红书文案** 进入文案生成器页面。

**访问路径：**
- URL: `http://你的域名/admin/xiaohongshu`
- 或点击管理后台侧边栏的"小红书文案"菜单

### 2. 基本使用流程

#### 步骤1：选择模板或输入提示词

**方式A：使用快捷模板（推荐）**
- 页面左侧提供了5个预设模板：
  - **故事向**：通过真实社恐瞬间引发共鸣
  - **测评实录**：展示完整测评体验过程
  - **干货教育**：科普知识+自测引导
  - **FAQ答疑**：常见问题解答
  - **UGC征集**：征集真实社恐瞬间

点击任意模板按钮，系统会自动填充该模板的提示词到输入框。

**方式B：自定义提示词**
- 在"自定义提示词"输入框中输入你的要求
- 例如：
  ```
  生成一篇关于开学季的故事向笔记
  重点突出AI个性化分析功能
  面向大学新生群体
  ```

**提示：** 如果留空输入框，系统会使用选中模板的默认提示词。

#### 步骤2：生成文案

点击 **✨ 生成文案** 按钮，等待AI生成（通常需要10-30秒）。

生成过程中按钮会显示"⏳ 生成中..."，请耐心等待。

#### 步骤3：查看并复制结果

生成成功后，右侧会展示完整的文案内容，包括：

- **📌 标题**：吸引眼球的标题（15-30字）
- **🎯 首屏三行**：开篇引入，直击痛点
- **📄 正文内容**：300-500字的主体内容
- **📢 行动号召**：引导用户评论/收藏的CTA
- **🏷️ 话题标签**：8-12个相关标签
- **🎨 首图建议**：配图创意提示

每个部分都有独立的 **📋 复制** 按钮，点击即可复制该部分内容。

#### 步骤4：一键复制完整文案

点击底部的 **📑 一键复制完整文案** 按钮，可以将所有内容（含标注）一次性复制，方便整理。

#### 步骤5：发布到小红书

1. 打开小红书APP，点击"+"创建笔记
2. 粘贴标题到标题栏
3. 粘贴正文内容到正文区域
4. 添加话题标签（从复制的标签中选择）
5. 根据首图建议制作或选择配图
6. 发布笔记

## 功能特点

### ✨ 核心优势

1. **专业内置提示词**
   - 基于产品实际情况和小红书爆款规律设计
   - 包含产品核心卖点、目标用户、文案要求等信息

2. **5种快捷模板**
   - 覆盖不同文案类型和使用场景
   - 点击即用，无需思考

3. **JSON结构化输出**
   - 标题、正文、标签等分类清晰
   - 便于分别复制和编辑

4. **一键复制**
   - 每个部分独立复制
   - 支持一键复制完整文案

5. **移动端友好**
   - 响应式设计
   - 手机也能快速生成文案

重点突出"社交反刍"和"预期焦虑"维度

**特定节点：**

**功能突出：**
```

展示从第33题预生成到提交秒开报告的体验
```

### 🎯 最佳实践

1. **先用模板熟悉**
   - 新手建议先使用5个快捷模板生成几篇，了解AI的输出风格

2. **组合使用**
   - 选择模板后，可以在提示词框中补充具体要求
   - 例如：选择"故事向"模板，然后加上"面向大学新生"

3. **多生成几版**
   - 同一个提示词可以多次生成，选择最佳版本
   - AI每次生成的结果会有差异

4. **编辑优化**
   - 生成后可以根据需要微调文字
   - 添加你的网站链接和专属引导语

5. **A/B测试**
   - 生成多个标题，测试哪个点击率更高
   - 对比不同风格的文案效果

## 常见问题

### Q1: 生成需要多久？
**A:** 通常10-30秒。如果超过1分钟可能是网络问题或API限流，可以重试。

### Q2: 生成失败怎么办？
**A:** 检查以下几点：
1. 确认后端服务是否启动（`backend`目录运行`npm run dev`）
2. 检查`.env`文件中`CLAUDE_API_KEY`是否配置正确
3. 查看浏览器控制台和后端日志的错误信息
4. 确认API余额是否充足

### Q3: 可以保存生成的文案吗？
**A:** 当前版本仅支持实时生成和复制，不保存历史记录。建议复制后粘贴到本地文档保存。

### Q4: 生成的标签太多怎么办？
**A:** 小红书笔记最多支持20个标签，你可以从生成的8-12个标签中选择最相关的使用。

### Q5: 生成的内容可以直接发布吗？
**A:** 建议先审阅内容，根据实际情况微调后再发布。特别注意：
- 添加你的网站实际访问链接
- 调整CTA引导语
- 确保符合小红书社区规范

### Q6: 如何修改内置提示词？
**A:** 编辑文件 `backend/routes/xiaohongshuRoutes.js`，找到 `SYSTEM_PROMPT` 常量进行修改。

## 技术说明

### API接口

**生成文案：**
```
POST /api/admin/xiaohongshu/generate
Headers: Authorization: Bearer {token}
Body: {
  "userPrompt": "用户提示词（可选）",
  "contentType": "模板类型（story/review/tutorial/faq/ugc）"
}
```

**获取模板：**
```
GET /api/admin/xiaohongshu/templates
Headers: Authorization: Bearer {token}
```

### 返回格式

```json
{
  "success": true,
  "data": {
    "title": "标题文字",
    "opening": ["首屏第一行", "首屏第二行", "首屏第三行"],
    "content": "正文内容...",
    "cta": "行动号召文字",
    "tags": ["标签1", "标签2", ...],
    "imageIdea": "首图建议描述"
  }
}
```
