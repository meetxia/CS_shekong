# 快速开始 - AI个性化分析功能

## 📦 已完成的集成

✅ 所有AI功能已经集成完毕，**开箱即用**！

## 🚀 启动项目

### 1. 安装依赖（如果还没安装）

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器会在 `http://localhost:5173` 启动（或显示的其他端口）

### 3. 开始体验

打开浏览器，访问首页，开始测评即可！

## ✨ 无需额外配置

AI功能已经配置好默认值，包括：

- ✅ **API Key**：已内置可用的API密钥
- ✅ **API URL**：`https://dpapi.cn/v1/chat/completions`
- ✅ **AI Model**：`claude-4.5-sonnet`

你**不需要**做任何配置，直接使用即可！

## 🎯 体验AI功能

### 快速测试流程

1. **填写基础信息**
   - 选择年龄、性别、职业、社交频率

2. **开始答题**
   - 认真回答35道题目

3. **观察预生成**（第33题）
   - 打开控制台（F12）
   - 答到第33题时，会看到：`🚀 提前开始AI报告生成...`
   - 继续答完第34、35题

4. **提交测评**
   - 点击"提交测评"按钮
   - 控制台显示：`⚡ 使用预生成的AI报告，秒开！`
   - 报告<1秒打开

5. **查看AI分析**
   - 在"你的社恐类型"部分看到 `🤖 AI智能分析` 标识
   - 查看个性化的类型名称、核心特征、心理根源分析

## 🔍 验证AI是否工作

### 方法1：查看AI标识

在报告页面，"你的社恐类型"标题旁边会显示：

```
🤖 AI智能分析
```

### 方法2：查看控制台日志

打开浏览器控制台（F12），成功时会看到：

```
🚀 提前开始AI报告生成...
🤖 正在使用AI生成个性化社恐类型分析...
✅ AI报告预生成完成！
⚡ 使用预生成的AI报告，秒开！
✅ AI生成成功，使用AI个性化分析
```

### 方法3：查看类型名称

AI生成的类型名称通常更加个性化和新颖，例如：

- "脑内彩排型社恐"
- "自我审查型社恐"
- "情绪雷达过载型"
- "评价敏感型社恐"

而不是简单的"预演型社恐"、"回避型社恐"等。

## 📊 性能指标

正常情况下，你会体验到：

- **预生成触发**：第33题作答时自动开始
- **生成完成**：答完35题前（约5-10秒内）
- **报告打开**：提交后<1秒即可查看
- **无感等待**：几乎感觉不到任何延迟

## 🛡️ 降级保护

如果AI服务暂时不可用（网络问题、API限额等），系统会自动降级：

```
🤖 正在使用AI生成个性化社恐类型分析...
❌ AI生成失败: ...
⚠️ AI生成失败，使用增强规则生成
```

- 使用**本地增强规则**生成（26种个性化类型）
- 不会显示AI标识
- 但仍然是高质量的个性化分析

## 🔧 可选：自定义配置

如果你想使用自己的API Key，可以创建 `.env` 文件：

```bash
# .env
VITE_AI_API_KEY=your-api-key-here
VITE_AI_API_URL=https://dpapi.cn/v1/chat/completions
VITE_AI_MODEL=claude-4.5-sonnet
```

然后重启开发服务器。

## ❓ 常见问题

### Q: 如何知道AI功能是否正常工作？

A: 三个标志：
1. 报告显示 `🤖 AI智能分析` 标识
2. 控制台显示"AI生成成功"
3. 类型名称个性化且新颖

### Q: 为什么有时没有AI标识？

A: 可能的原因：
- AI生成失败（网络、API限额等）
- 系统自动降级到本地增强规则
- 你仍然会获得高质量分析，只是不是AI生成的

### Q: 预生成什么时候触发？

A: 在第33题（倒数第3题）作答时自动触发。如果你跳题或答题速度很快，可能来不及预生成，会改为实时生成。

### Q: 实时生成需要多久？

A: 如果没有命中预生成，实时生成大约需要3-5秒。这时会显示"正在生成报告..."的提示。

### Q: API Key会用完吗？

A: 内置的API Key有充足额度，正常使用不会有问题。如果担心，可以使用自己的API Key。
