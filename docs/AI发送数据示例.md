# AI发送数据示例

## 📊 发送给AI的数据内容

### ✅ 会发送的信息

发送给AI API的提示词包含以下**匿名化的统计信息**：

#### 1. 用户基本信息（已脱敏）
```
年龄段: 18-22岁（大学生）
性别: 女
职业: 学生
社交频率: 1-2次/周
```

**🔒 隐私说明**：
- ✅ 只发送年龄**段**（如"18-22岁"），不是具体年龄
- ✅ 只发送性别分类（男/女/其他）
- ✅ 只发送职业**类别**（学生/职场人等），不是具体职位
- ✅ 只发送社交频率**等级**，不是具体活动内容

#### 2. 测评结果数据
```
总分: 65/100
等级: 中度社交焦虑
初步类型: 预演型社恐

维度得分详情:
社交场景恐惧: 18/25 (72%) - 偏高
回避行为程度: 15/25 (60%) - 中高
预期焦虑强度: 20/25 (80%) - 偏高
负面评价恐惧: 17/25 (68%) - 中高
社交后反刍: 19/25 (76%) - 偏高
功能损害程度: 14/25 (56%) - 中等

高焦虑题目: Q1, Q3, Q6, Q7, Q13, Q18, Q24...
```

**🔒 隐私说明**：
- ✅ 只发送分数和统计结果
- ✅ 只发送题目编号（Q1, Q2...），不发送具体题目内容
- ❌ **不发送**具体答案内容
- ❌ **不发送**用户填写的任何文字

#### 3. AI任务要求
```
请生成以下内容（JSON格式）：
1. 个性化社恐类型名称（15字以内）
2. 英文名称
3. 核心特征（3-5条）
4. 心理根源分析（2-3个维度）
5. 正向重构（60-80字）
```

---

### ❌ 不会发送的信息

以下信息**绝对不会**发送给AI：

1. ❌ 用户的真实姓名
2. ❌ 联系方式（手机号、邮箱等）
3. ❌ IP地址、设备信息
4. ❌ 具体的答题内容（只发送分数）
5. ❌ 任何可识别个人身份的信息

## 🔐 隐私保护措施

### 1. 数据最小化原则
- 只发送分析必需的统计数据
- 不发送任何可识别个人身份的信息

### 2. 数据脱敏处理
- 年龄 → 年龄段（18-22岁）
- 职业 → 职业类别（学生）
- 答案 → 分数统计（只发送题号，不发送内容）

### 3. 本地优先
- 数据存储在浏览器本地（localStorage）
- 不会上传到任何服务器
- 用户可以随时清除数据

### 4. API安全
- 使用HTTPS加密传输
- API Key配置在环境变量中
- 超时自动中断（30秒）

## 📊 数据流转示意图

```
用户答题
  ↓
本地计算分数（浏览器）
  ↓
生成统计数据（匿名化）
  ↓
发送给AI API（只包含统计数据）
  ↓
AI返回个性化分析
  ↓
显示在报告页面
  ↓
保存在本地浏览器（不上传服务器）
```

## ✅ 总结

**发送给AI的数据是**：
- ✅ 匿名的统计信息（年龄段、性别、职业类别）
- ✅ 测评分数和维度得分
- ✅ 题目编号（不含具体内容）

**不发送的数据**：
- ❌ 任何个人身份信息
- ❌ 具体的答题内容
- ❌ 联系方式
- ❌ 设备信息

**这种设计确保**：
1. AI能够根据用户的测评模式生成个性化分析
2. 用户的隐私得到充分保护
3. 即使AI API被攻击，也无法追溯到具体个人
