# 社恐测评MVP - 快速测试清单 ✅

## 🎯 测试入口

开发服务器已启动，访问：`http://localhost:3000`

## 📝 快速测试流程（5分钟）

### 1️⃣ 激活页测试（1分钟）

访问首页后：

- [ ] 输入框显示正常，placeholder文字清晰
- [ ] 尝试输入：`test2024abcd`（自动转换为 `TEST-2024-ABCD`）
- [ ] 观察按钮状态：格式正确后按钮变为可点击
- [ ] 点击"开始测评"，等待验证（约1秒）
- [ ] 看到绿色提示"激活成功！"
- [ ] 页面淡出后自动跳转到测评页

**测试激活码示例：**
```
TEST-2024-ABCD
DEMO-1234-5678
ABCD-EFGH-1234
```

### 2️⃣ 测评页测试（3分钟）

- [ ] 顶部显示"第1/共30题 3%"
- [ ] 进度条为主题色（灰茶红）
- [ ] 题目文字清晰可读
- [ ] 5个选项卡片样式美观
- [ ] 点击第一个选项，观察：
  - 选项高亮显示
  - 出现"✓"标记
  - 0.5秒后自动跳转到第2题
- [ ] 点击"上一题"回到第1题，答案保持选中
- [ ] 快速完成前10题，观察是否出现"已完成33%，继续加油！"提示
- [ ] 可以选择快速完成剩余20题，或关闭浏览器再打开测试断点续答功能
- [ ] 完成第30题后，按钮变为"提交测评"
- [ ] 点击提交，看到loading状态"生成报告中..."
- [ ] 等待2秒后自动跳转到报告页

### 3️⃣ 报告页测试（1分钟）

- [ ] 顶部导航栏正常显示
- [ ] 总分卡片（渐变背景）显示分数和等级
- [ ] 等级图示清晰，能看到自己的位置标记
- [ ] 六维度雷达图正常渲染（六边形）
- [ ] 滚动查看维度详解，每个维度有进度条和分数
- [ ] 社恐类型诊断卡片显示完整
- [ ] 改善建议内容丰富
- [ ] 点击右上角"切换配色"按钮：
  - [ ] 弹出配色选择器
  - [ ] 尝试切换到"芩江初雪（浅色）"（绿色系）
  - [ ] 页面颜色立即切换
  - [ ] 雷达图颜色同步更新
  - [ ] 再切换到"雪尽霜余（深色）"测试深色模式
- [ ] 点击左上角"返回"回到测评页
