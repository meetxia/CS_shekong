# 报告页雷达图深浅色适配修复

## 背景
报告页（`src/views/ReportPage.vue`）的雷达图在浅色模式下文字对比度不足，出现看不清的问题。

## 变更点
- 使用全局主题变量 `--text-title`、`--text-secondary`、`--border` 动态设置 ECharts 文本与网格颜色。
- 增强浅色模式下标签文字可读性：提高 `fontWeight`、增加 `textShadow`。
- 使用主题化的 `gridColor` 提升网格与背景的对比度。
- 新增 `tooltip` 并适配深浅色样式。
- 监听 `body.class` 变化（主题切换）并自动重绘雷达图；补充窗口 `resize` 处理与卸载清理。

## 涉及文件
- `src/views/ReportPage.vue`

## 关键代码（节选）
```12:32:src/views/ReportPage.vue
const computedStyle = getComputedStyle(document.documentElement)
const textTitle = computedStyle.getPropertyValue('--text-title').trim()
const borderColorVar = computedStyle.getPropertyValue('--border').trim()
const isDark = document.body.className.includes('-dark')

const gridColor = borderColorVar || (isDark ? 'rgba(255,255,255,0.18)' : 'rgba(0,0,0,0.15)')
const labelColor = textTitle || (isDark ? '#F5E6D3' : '#1A1A1A')

name: {
  textStyle: {
    color: labelColor,
    fontSize: 13,
    fontWeight: 500,
    textShadowColor: isDark ? 'rgba(0,0,0,0.2)' : 'rgba(255,255,255,0.6)',
    textShadowBlur: 2,
    textShadowOffsetY: 1
  }
},
```

## 验证方式
1. 在浅色模式下打开报告页，确认雷达图维度标签、网格线对比度清晰；Tooltip 文本可读。
2. 切换至深色模式，确认颜色自适应且无重影过亮问题。
3. 浏览器缩放/窗口大小变化，确认雷达图自适应。

- 风险较低，仅影响报告页雷达图渲染；如需回滚，可撤销本次对 `ReportPage.vue` 的改动。
